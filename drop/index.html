Claude Code Skill

PM Weekly Product Pulse

Weekly Slack intelligence for Product Managers — top questions, pain points, feature requests, and sentiment delivered as a Canvas.

What it does

  • 🔍Scans public channels + your DMs for the past 30 days, anchored to your product
  • 📊Clusters findings into top questions, pain points, and feature requests
  • 💬Surfaces cross-product friction and domain-level discussions
  • 📋Posts a structured Canvas to a Slack channel of your choice
  • 🧠Learns your preferences over time — tell it what to exclude or reclassify

Install in 2 steps

  1. 1
    Add the marketplace and install

    In Claude Code, run these two commands:

    /plugin marketplace add https://raw.githubusercontent.com/SavannahforMollie/pm-weekly-product-pulse/main/.claude-plugin/marketplace.json click to copy

    /plugin install pm-weekly-product-pulse click to copy
  2. 2
    Connect Slack

    The skill reads your Slack messages, so Claude Code needs access:

    /plugin install slack click to copy
    Select user scope and complete the OAuth flow. Your own Slack permissions apply — you can only access channels you already have access to.

Run it

Trigger it by saying something like "weekly product pulse for Resell Pricing", or use the slash command:

/pm-weekly-product-pulse:pm-weekly-product-pulse Resell Pricing post in #your-channel click to copy
First run will ask you to set defaults (product, channel). After that it remembers your preferences.
